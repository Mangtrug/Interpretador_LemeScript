Introdução ao LemeScript

LemeScript é uma linguagem de programação simples que se baseia na sintaxe do JavaScript. É projetada para ser fácil de entender e usar, com comandos semelhantes aos do português.

Principais Palavras-Chave

  • se - Estrutura condicional (equivalente ao if em JavaScript).
  • entao - Define o bloco de código a ser executado se a condição for verdadeira.
  • fim - Finaliza uma estrutura condicional ou um bloco de código.
  • log - Imprime informações no console (equivalente ao console.log em JavaScript).
  • entrada - Captura a entrada do usuário (equivalente ao prompt em JavaScript).

Exemplos de Código

Exemplo 1: Estrutura Condicional


se x > 10 {
    log("x é maior que 10");
}
fim

Exemplo 2: Entrada do Usuário


entrada = entrada("Digite seu nome:");
log("Olá, " + entrada);

Exemplo 3: Estrutura Condicional com Entrada


entrada = entrada("Digite um número:");

se entrada > 100 {
    log("O número é maior que 100");
}
fim
